# Privacy Policy for Bookmark Import Export Manager

Bookmark Import Export Manager does not collect, transmit, sell, or share user data.

The extension works locally in the user's browser. It uses the Chrome bookmarks permission only to read, export, create, import, and delete bookmarks when the user requests those actions.

The downloads permission is used only to save exported bookmark files to the user's device.

The storage permission is used only to remember the user's interface language preference.

Bookmark data, exported files, and language preferences are not sent to any server controlled by the developer.
